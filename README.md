# College-project-DB
Contains all the tables and the setup file for the NBA DB
Steps:
  1) Clone the repository
  2) git clone https://github.com/<your-username>/college-project-db.git
  3) Open MySQL Workbench and connect to your MySQL server.
  4) Open the file setup.sql.
  5) Run the script (⚡ Execute).

This will:
  1) Create the database college_db
  2) Create all tables.
